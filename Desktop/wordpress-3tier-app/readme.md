# Terraform WordPress deployment
## Description:
This TF script will deploy a single region highly available WordPress site with RDS, EC2 and VPC into AWS account

### Networks to be provisioned:
- 1 VPC 
- 2 Database subnets 
- 1  Web subnets 
- 2  public subnets 

### Resources:
- 1 NLB
- 2 web servers 
- 1 RDS instance (MySQL 5.7)
